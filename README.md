# DIAL-for-Cross-Domain-UAV-RFFI

This repository provides the implementation of our paper on Robust Cross-Domain Drone RFFI Method Using Domain-Invariant Adversarial Learning and Manifold Regularization (TCCN Accept).

# Overview

Unmanned aerial vehicle (UAV) identification based on radio frequency fingerprints is an important technique for physical-layer security. However, most existing methods assume that training and testing data are collected under the same conditions, which often causes performance degradation when the environment changes.
To address this issue, this work proposes a robust cross-domain UAV RFFI method by combining *domain-invariant adversarial learning* and *manifold regularization*. The method is designed to learn class-discriminative yet domain-invariant features, while improving generalization through interpolation in the hidden feature space.
